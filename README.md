FIX::Lite
========

DESCRIPTION

Simple FIX (Financial Information eXchange) protocol module

INSTALLATION

To install this module type the following:

   perl Makefile.PL
   make
   make test
   make install

DEPENDENCIES

This module requires these other modules and libraries:

IO::Socket
IO::Select
Time::HiRes
